# Segmentation Fault

## 개념

- 잘못된 메모리를 참조했을 때 발생하는 오류
- 운영 체제 수준에서 막아주는 에러

## 원인

- 잘못된 메모리 참조 때문
- 건드리지 말아야 할 곳을 건드려서 발생
- 주로 Null 로 설정된 영역을 건드리거나, 할당받은 메모리 공간을 넘은 곳을 건드렸을 때 발생
